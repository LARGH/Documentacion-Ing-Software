\subsection{IU4.0 Pantalla de Perfil del representante.}

\subsubsection{Objetivo}
	El representante puede modificar sus datos personales en caso de que lo requiera.

\subsubsection{Diseño}
\IUfig[.6]{gui/IU4_0}{IU4.0}{Pantalla de Perfil del representante.}
\IUfig[.6]{gui/IU4_1}{IU4.0}{Pantalla de Perfil del representante.}
  El actor debe puede modificar sus datos personales como si fuerala primera ve que se registra.


\subsubsection{Opciones}
\begin{itemize}
	\item \IUbutton{Examinar}: Despliega una ventana para elegir una imagen.
	\item \IUbutton{Actualizar mis datos}: Guarda los cambios realizados en el perfil del representante.
  \end{itemize}
